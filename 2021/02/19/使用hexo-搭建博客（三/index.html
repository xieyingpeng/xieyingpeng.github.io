1、GitHub创建个人仓库

首先，你需要有一个自己的 Github 账号，没有的可以去注册。

注册完登录后，我们去New repository ，新建仓库。

  • 注意：创建一个和你用户名相同的仓库，后面加.github.io，只有这样，将来要部署到GitHub page的时候才会被识别，也就是xxxx.github.io，其中 xxx 就是你注册GitHub的用户名。

点击 create repository 即可。

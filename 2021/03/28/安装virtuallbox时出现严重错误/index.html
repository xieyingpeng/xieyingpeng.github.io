安装virtualbox时出现Installation failed! Error错误

1.问题：

电脑操作系统为Windows10 64位
软件VirtualBox 5.2.32和VirtualBox 6.0.10都安装过，
提示：
但是中途弹出框提示：

安装virtualbox时安装程序出现严重错误

￼

2.解决方法1：

在我的电脑 -> 管理 -> 服务和应用程序 -> 服务 开启如下两个服务
（ 1），Device Install Service.
（2）, Device Setup Manager.

3.解决方法2：

当地一种方法尝试多次不生效之后，改变策略进行安装目录的修改。
因之前想安装在D盘的文件目录下，导致每次出现问题，后修改默认到C盘目录下，安装成功。

如果方法1 2都没有安装成功试一下下面这个方法

安装在c盘需要权限，所以用管理员身份打开然后就安装完成了。

4.解决方法3：

如果以上方法都没有成功

那么我们需要下载一个软件Ccleaner清理注册表，因为我们安装了注册表有安装残留。Ccleaner清理注册表，我的清理完，再开启两项服务，成功安装VirtualBox。

5.软件下载和安装使用

（1）下载链接：https://pan.baidu.com/s/19N6D0-8n6MAGvAnl_FOiXQ
提取码：32f3

（2）使用步骤：

安装完成Ccleaner，双击“Ccleaner”打开该应用，在CCleaner如图界面，会检查版本更新，单击“确定”，进入下一步
￼
进入“CCleaner注册”界面，不输入许可证密钥，单击“取消”进入应用
￼
在CCleaner应用程序主界面，第一步单击左侧“注册表”，再点击“扫描问题”，扫描完成，修复选定的问题，点击全部修复即可。

￼
扫描完，修复“全部选定”的问题即可，在开启服务安装。

MySQL的启动和登录

1.启动MySQL服务

（1）cmd控制台启动。这种方式非常简单，net start mysql57表示启动MySQL服务，net stop mysql57表示关闭MySQL服务。
（2）手动启动。执行“开始”“设置”“控制面板”“管理工具”“服务”命令进行设置，（执行“开始”“运行”命令并输入services.msc后按Enter键也可以弹出）。

（3）在图中可以更改MySQL服务的启动类型，选中MySQL57服务项右击，在弹出的快捷菜单中选择“属性”命令，弹出如图所示的对话框。

（4）从图中可以看到，可以更改服务状态为“停止”、“暂停” 和“ 恢复”，还可以设置服务的启动类型。在“启动类型”下拉列表框中可以选择“自动”、“手动”和“已禁用”选项，说明如下。

2.登录MySQL数据库

（1）MySQL服务启动后，可以通过客户端来登录MySQL数据库。Windows操作系统下有两种登录MySQL数据库的方式：
（2）执行cmd命令，在打开的DOS窗口中以命令行的方式登录MySQL数据库；
在MySQL客户端直接登录数据库。

(3) 在DOS窗口中登录MySQL数据库。通过DOS窗口登录MySQL数据库执行语句时，可以执行“开始” “运行”命令，在弹出的对话框中输入cmd后按Enter键，即可进入DOS窗口。
登录成功后会出现“Welcome to the MySQL monitor”的欢迎语。

C:\Users\Administrator>net start mysql57
请求的服务已经启动。
C:\Users\Administrator>mysql -u root -p

上述执行语句的代码中，-u后面紧跟着数据库的用户名，此处使用root用户进行登录；-p表示用户密码，按Enter键输入密码，输入的密码使用星号(*)表示。

(4) MySQL客户端登录数据库。在Windows7操作系统中，执行“开始”  “所有程序” MySQLMySQL 5.7目录。
MySQL Command Line Client MySQL
Command Line Client-Unicode`
它们都是MySQL客户端的命令行工具，也可以称为MySQL的DOS窗口或控制台。通过在控制台中执行语句可以登录MySQL数据库，然后执行其他的相关SQL语句进行操作。

3.DOS窗口和控制台包含一些说明性的语句如下：

（1）Commands end with ; or \g：说明MySQL控制台下的命令是以分号(;)或“\g”来结束的，遇到这个结束符就开始执行命令。
（2）Your MySQL connection id is 5：id表示MySQL数据库的连接次数，如果数据库是新安装的，且是第一次登录，则显示1。如果安装成功后已经登录过，将会显示其他的数字。
（3）Server version：Server version之后的内容表示当前数据库版本，这里安装的版本是5.7.17-enterprise-commercial-advanced。
（4）Type 'help;' or '\h' for help：表示输入“help;”或者\h可以看到帮助信息。
（5）Type ‘\c’ to clear the current input statement：表示遇到\c就清除当前输入语句。

（6）mysql> select now();
+-——————–+ | now() | +———————+ | 2018-01-10 10:10:00 | +———————+ 1 row in set (0.05 sec) mysql>

（7）例如，登录MySQL数据库成功后可以直接输入“help;”或\h查看帮助信息，直接在控制台中输入“help;”语句按Enter键，输出结果如下：
mysql> help
……帮助信息
Mysql>
